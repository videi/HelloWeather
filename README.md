# Hello Weather (Test Project) 🌦️

Тестовое мобильное приложение для отображения прогноза погоды, разработанное в рамках обучения архитектуре и работе с сетевыми запросами.

## О проекте
Приложение запрашивает текущую геолокацию пользователя и выводит детальную информацию о погоде:
- Текущая температура, город и состояние (локализовано через API).
- Почасовой прогноз на ближайшие сутки.
- Прогноз на 3 дня с минимальной и максимальной температурой.
